Scrolling Example

This Demo is created to show the concept of continuos scrolling of pages. The intent is to create a tool that can help reading large files.

Tool takes two parameters:

  • Page Cache: Amount of data that can be fetched from the server. Larger sets will create jitter in UI.
  • Cache Limit: Number of pages container can store.

Following Parameters are just for illustration purpose

  • First Page: First cached page number.
  • Last Page: Last cached page number.
